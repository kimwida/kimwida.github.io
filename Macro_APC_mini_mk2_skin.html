Make a Colored AKAI APC mini MK2's Skin. It's Not MIDI Feedback!

How to Use
***To use this macro, you have to use over T2. Titan one and T1 users can't use this macro***
1. Make your own skin with the drop boxes.
2. Push "Make a Macro Script" Button.
3. Copy and paste the context to notepad.
4. Save as "akai_apc_mini_MK2_skin.xml"
5. Move the file to Titan Macro folder. Usually it is located in "%userprofile%\documents\titan\macros"
6. Restart Titan.
7. Search the macro in "Show Library" with "Make APC mini MK2 skin" keywords.
8. Move the macro and run it.
