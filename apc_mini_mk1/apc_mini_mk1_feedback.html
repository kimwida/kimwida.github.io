== How to run midimonster with APC mini mk2 ==
 1. Extract the zip file to "c:\midimonster".
2. Edit "c:\midimonster\monster.cfg" with your favorite text editor
3. Change "2.1.1.19" to your PC's ip address.



3. Run command prompt.
4. "cd c:\midimonster" and enter
5. "midimonster" and enter



== How to change the skin of APC mini mk2 ==
1. Edit "c:\midimonster\Variable.lua" with your favorite text editor.
2. "Skin" array has 9 rows and 9 columns like APC mini mk2's buttons.
3. Change colors in "Skin" array with the colors you want.
4. 9th column and 9th row in "Skin" array have only one color. (Green or Red)
5. And the far right and bottom button of APC mini mk2 have no led. So it has no color.





5. You can make a button blinking by editing "Pushed" array.
6. If you want a button blinking, just change "No" to "Yes".



== How to make MIDI feedback of APC mini mk2 working ==
1. Change "No" to "Yes" in "Feedback" array in "Variable.lua".



== How to make playback group MIDI feedback of APC mini mk2 working ==
1. Change "0" to any number of playback group.



2. If you have 3 playbacks in first playback group, you put "1" in the position of the playbacks.
3. If you have 5 playbacks in second playback group, you put "2" in the position of the playbacks.

1. Firstly run MIDI Monster. Secondly run Avolites Titan.
2. Windows can run only one MIDI program. So if you run MIDI Monster in advance to Titan, you can use MIDI Monster with.
